# 🧪 OrangeHRM Automation

![CI](https://github.com/Manuel1286/orangehrm-automation/actions/workflows/test.yml/badge.svg)

Automatización de pruebas web para OrangeHRM usando **Serenity BDD** y el patrón **Screenplay**.

---

## 🚀 Tecnologías

- ☕ Java 17
- 🧪 Serenity BDD
- 🧱 Screenplay Pattern
- 🌐 Selenium WebDriver
- ⚙️ Maven
- 🔄 GitHub Actions (CI)

## 🧠 Patrón de Diseño

Este proyecto implementa el patrón **Screenplay**, el cual permite:

- Separación clara de responsabilidades
- Código reutilizable y mantenible
- Escenarios más legibles
